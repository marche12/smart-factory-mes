📝 기초 데이터 등록

2
📋 작업지시서 작성
거래처 선택 → 품목 선택 → 공정 설정 → 저장
인쇄 공정 자동 "진행중" 생산현황판 자동 등록 생산계획 자동 등록 알림벨 자동 갱신 작업자 대기열 갱신
3
🖨️ 인쇄 외주 진행
인쇄소에서 인쇄 → 자사 입고 → 관리자 입고 확인
인쇄 공정 "완료" 처리 다음 공정 대기열 등록 작업 이력(hist) 기록 생산현황판 갱신

🔄 내부 공정 순환 (코팅 → 합지 → 톰슨 → 접착)

▶️ 작업자 [시작] 클릭

✅ 작업자 [완료] 클릭
